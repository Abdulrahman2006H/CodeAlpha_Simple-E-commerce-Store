# 📚 BookMuse - Online Book Store

BookMuse is a modern online book store web application built with Angular for the frontend and ASP.NET Core Web API for the backend.  
The project allows users to browse books, view book details, add books to cart, register/login, place orders, and view their previous orders.

---

## 🚀 Features

### Frontend
- Modern responsive book store UI
- Home page with featured books and categories
- Books listing page
- Search and category filtering
- Book details page
- Shopping cart
- Add/remove/increase/decrease cart items
- Checkout page
- User login and registration pages
- My Orders page
- Book cover images loaded from backend static files

### Backend
- ASP.NET Core Web API
- SQL Server database
- Entity Framework Core
- Books CRUD API
- User authentication API
- Orders processing API
- Order items saved in database
- User orders endpoint
- Static files support for book images
